今回は、将棋ウォーズの棋譜をダウンロードする方法をご紹介します！
ダウンロードした棋譜は、KIFファイルとして将棋ソフト「ShogiGUI」への活用が可能となります。
# # 将棋ウォーズの棋譜をダウンロードする方法
# # # 将棋ウォーズ ブラウザ版にアクセス
①ブラウザを起動し、**将棋ウォーズ ブラウザ版**にアクセスします。
※既にログイン済である場合は、下図の画面は表示されません。「将棋ウォーズ ブラウザ版ログイン」はスキップしてください。

# # # 将棋ウォーズ ブラウザ版ログイン
①画面内をクリックすると、ログイン画面が表示されます。



②ユーザーID、パスワードを入力し、「ログインする」をクリックします。



③ログインに成功したら「OK」をクリックします。


# # # ダウンロード対象の対局を選択
①将棋ウォーズ初期画面で、「マイページ」をクリックします。

↓

<img src="/images/blog/将棋ウォーズ新規画面.jpg" alt="" style="max-width: 100%; height: auto; margin: 1rem 0; border-radius: 8px; border: 1px solid #e2e8f0;" />
アカウント情報画面が表示されます。



②下にスクロールし、「対局結果・棋譜一覧」をクリックします。

↓

<img src="/images/blog/マイページ初期画面.jpg" alt="" style="max-width: 100%; height: auto; margin: 1rem 0; border-radius: 8px; border: 1px solid #e2e8f0;" />
対局結果・棋譜一覧画面が表示されます。



③ダウンロードしたい対局の「見る」ボタンをクリックします。

↓

<img src="/images/blog/マイページ初期画面入力.jpg" alt="" style="max-width: 100%; height: auto; margin: 1rem 0; border-radius: 8px; border: 1px solid #e2e8f0;" />
対局画面が表示されます。


# # # 棋譜のURLをコピー
①対局画面で「ツイート」をクリックします。

↓

<img src="/images/blog/マイページ初期画面入力ログインOK.jpg" alt="" style="max-width: 100%; height: auto; margin: 1rem 0; border-radius: 8px; border: 1px solid #e2e8f0;" />

「Twitterでリンクを共有する」画面が表示されます。
**※Twitterにログインする必要はありません。**



②「Twitterでリンクを共有する」画面の以下のキーを入力して、**テキストボックスの内容をすべてコピー**します。

「Ctrl + A」（全選択のショートカットキー）
↓

<img src="/images/blog/ログイン後初期画面.jpg" alt="" style="max-width: 100%; height: auto; margin: 1rem 0; border-radius: 8px; border: 1px solid #e2e8f0;" />
「Ctrl + C」（コピーのショートカットキー）


# # # 将棋ウォーズ棋譜保存サイトからダウンロード
①**将棋ウォーズ棋譜保存**にアクセスします。



②「Twitterでリンクを共有する」画面でコピーしたテキストボックスの内容を貼り付けます。



③検索アイコンをクリックし、検索を実行します。



④「保存」が表示されますので、「保存」を右クリックし、「対象をファイルに保存」をクリックします。



⑤「名前を付けて保存」ダイアログ画面が表示されますので、任意のフォルダを指定して「保存」ボタンをクリックします。



以上となります！
